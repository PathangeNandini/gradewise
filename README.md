## 📖 Overview

GradeWise is a web application built to simplify GPA and CGPA calculations for VIT students. Instead of manually calculating grades, students can instantly calculate their Semester GPA, predict future CGPA, estimate CGPA after grade improvements, and upload their VTOP grade history for quick academic analysis.

---

## ✨ Features

- 📊 Semester GPA (SGPA) Calculator
- 🎯 Future CGPA Predictor
- 🔄 CGPA After Grade Improvement Calculator
- 📂 Upload Grade History
- 🌙 Light & Dark Theme
- 📱 Responsive Design (Mobile, Tablet & Desktop)
- ⚡ Fast Real-time Calculations
- 🎨 Clean & Modern UI

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/PathangeNandini/gradewise.git
```

### Navigate

```bash
cd gradewise
```

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Visit:

```
http://localhost:5173
```
